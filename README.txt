Product Lister for Etsy allows user to sync their WooCommerce store and its products to the Etsy Marketplace.

== Description ==

**Product Lister for Etsy** allows user to integrate WooCommerce with Etsy, allowing you to sell on Etsy easily with your WooCommerce store and manage everything from one place.

Product Lister for Etsy 3.1.2 is available now with all new features suggested by Wordpress Community. We are getting huge response from the community which keep motivating us to add more useful feature to this plugin and make life easier than never before.

<strong> For advance features of **Product Lister for Etsy**, upgrade to the premium version of <a href="https://woocommerce.com/products/etsy-integration-for-woocommerce/">Etsy Integration For WooCommerce</a></strong>. 


##MAJOR FEATURES##

* **List Single Products on Etsy:** Enable the ability to easily upload individual products to the Etsy marketplace.

* **List Variable Products on Etsy:** Facilitate the listing of products with multiple variations on Etsy.

* **Automatic Price and Inventory Synchronization:** Develop a feature that ensures seamless synchronization of prices and inventory levels between Etsy and WooCommerce.

* **Category Mapping:** Implement a category mapping system that allows products to be assigned to the appropriate categories on Etsy, streamlining the listing process.

* **Profile Settings Management:** Create a user-friendly interface for users to manage and customize their profiles and settings.

* **Bulk Product Sync (Up to 100 products):** Allow users to effortlessly transfer a significant number of products from WooCommerce to Etsy, up to a limit of 100 products at once.

* **Timline Activity:** Allow users to check all the activities regarding Inventory, Products and Cron execution in timeline section.

##Etsy Integration For WooCommerce Premium Version##

Moreover, you can also import Etsy product listings to your WooCommerce store. With the integration solution in place, sellers can efficiently manage their online presence by listing products on both platforms simultaneously and ensuring consistent inventory updates.

= Automatic installation =

In the search field type " Product Lister for Etsy" and click Search Plugins. Once you've found our  'Product Lister for Etsy' you can view details about it such as the point release, rating and description. Most importantly of course, you can install it by simply clicking "Install Now".

= Manual installation =

The manual installation method involves downloading the plugin and uploading it to your webserver via your favourite FTP application. The WordPress codex contains [instructions on how to do this here](http://codex.wordpress.org/Managing_Plugins#Manual_Plugin_Installation).
